## 💻 **How to Run**

To execute this part of the project, follow these steps:

1. **Download and Install Node.js**:
- Node.js official site: [Node](https://nodejs.org/)

2. **Navigate to the Project Directory**:
   ```bash
   cd '.\Lista 2\'
   ```

3. **Run the Project**:
   ```bash
   npm start
   ```

4. **Open in Browser**:
- Visit http://localhost:3000
